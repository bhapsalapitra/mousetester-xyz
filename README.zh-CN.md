## 测试工具

### 鼠标按键测试

检测全部 5 个鼠标按键（左键、右键、中键、后退、前进）、滚轮和传感器抖动。交互式鼠标图示会在你按下按键的瞬间高亮显示，是确认所有按键是否正常注册的最快方式。

### CPS 点击速度测试

测量你在 1 秒到 60 秒内的每秒点击次数（CPS）。工具会记录总点击数、平均 CPS 和峰值 CPS，适合测评连点（Jitter Clicking）、蝴蝶点击（Butterfly Clicking）、拖拽点击（Drag Clicking）等各种点击技巧。

### 双击检测

检测微动弹跳——最常见的鼠标故障。当微动老化时，按一次可能会注册为两次点击。工具会测量连续点击事件之间的时间间隔，并标记低于设定阈值的异常间隔，帮你判断微动是否需要更换或调整防抖设置。

### 回报率测试

鼠标回报率决定了它每秒向电脑报告位置的频率，单位为赫兹（Hz）。回报率越高，光标移动越流畅、响应越快。工具在你移动鼠标时捕获原始事件时间戳并计算实际回报率，让你验证鼠标是否真正达到了标称的 500Hz 或 1000Hz。

### DPI 精度测试

DPI（每英寸点数）决定了物理移动鼠标的距离与光标在屏幕上移动距离的对应关系。很多鼠标出厂时 DPI 档位不准，传感器也会随时间漂移。工具让你移动一段已知的物理距离，将预期像素位移与实际结果进行对比，揭示真实 DPI 及与设定值之间的偏差。

### 即将上线

- **延迟测试** — 测量从点击到屏幕响应的延迟
- **拖拽测试** — 评估拖拽操作的精度与流畅度
- **滚动测试** — 分析滚轮速度与步进一致性
